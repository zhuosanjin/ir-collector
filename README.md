# 主机侧应急响应信息采集工具
- 基于 Rust 开发轻量级主机取证工具，实现入侵事件中的快速信息采集
- 支持进程、网络连接、持久化后门、登录日志等6类关键数据自动收集

---

## 采集的信息模块

- [x] 系统基础信息
  - 主机名、操作系统版本、内核版本、运行时长、当前时间。
- [x] 进程模块
  - 当前所有运行进程，包括PID、父进程PPID、进程名、可执行文件路径、启动时间、命令行参数、对应用户。
- [x] 网络连接模块
  - 当前所有网络连接，包括本地IP、本地端口、远程IP、远程端口、协议、状态、对应进程PID。
- [ ] 用户与登录模块
  - 当前所有用户，包括用户名、UID、GID、主目录、登录状态、登录时间、登录方式。
- [ ] 持久化机制模块
  - Windows侧：注册表启动项（Run/RunOnce）、计划任务、服务列表、启动文件夹。
  - Linux侧：crontab（系统级+用户级）、systemd服务、/etc/init.d、~/.bashrc和/etc/profile里的异常注入。
- [ ] 文件系统模块
  - 最近N小时内被修改的文件、重点目录监控（temp、web目录）、具有SUID权限的可执行文件。

---

## 输出格式
- [ ] JSON格式：机器可读，方便后续接入SIEM或写脚本二次处理。
- [ ] Markdown格式：人类可读，直接在任何Markdown渲染器里看，层次清晰。
- [ ] HTML格式：带简单样式，双击就能在浏览器打开，适合直接给非技术的甲方看报告。
